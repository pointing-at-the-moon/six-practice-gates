# 附錄 C　名相對照

正文裡，凡遇梵文、巴利的名相，多半只在首次出現時，用白話夾注一句，之後便直接用漢語詞，不再標注那些帶符號的羅馬字。完整的羅馬轉寫，連同巴利（巴）與梵文（梵）的對照，都收在這裡。

想知道某個詞的原文怎麼寫、它本來是什麼意思的讀者，可以查這張表。不想理會這些的讀者，跳過它，一點不影響讀這本書——這正是把它收在書後的緣故。

（標「巴」者為巴利，「梵」者為梵文；同一詞兩傳統寫法不同者並列。）

---

## 一、三個骨幹詞，與它們的反面

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 施 | dāna（巴／梵） | 給出去的這個行為——看得見的那隻伸出去的手。 |
| 捨 | cāga（巴）／ tyāga（梵） | 心裡那一鬆——不再攥著「我的」。本書真正的主角。 |
| 供／供養 | yāga（巴／梵，祭儀語感）／ pūjā（梵，恭敬奉上） | 帶著敬意，把東西奉上；對象是福田（三寶與善人）。 |
| 慳 | macchera（巴）／ mātsarya（梵） | 心不肯讓「我的」變成「我的＋他的」——遇人乞時那一縮。捨的反面。 |

## 二、施的種類

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 財施 | āmisadāna | 把財物給出去。 |
| 法施 | dhammadāna（巴）／ dharmadāna（梵） | 把法給出去——說法、傳法、活出法。 |
| 無畏施 | abhayadāna | 把「不必怕」給出去——守住不殺、不盜，眾生便從你這裡得了安心。 |
| 恭敬施 | （prasāda-dāna，淨信之施） | 以一份恭敬、一份「你本來是佛」的肯定為施——常不輕的禮拜。 |
| 身（命）施 | dehadāna | 連身、連命都布施出去——藥王燃身的那一型。 |
| 隨喜 | anumodanā | 自己沒出力，只為別人的善事由衷歡喜——也是一份實實在在的施。 |

## 四、我執，與「空」

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 我所 | mamattā / mamāyita | 「我的」——處處要把世界劃進「我的」範圍的那個習慣。 |
| 我執 | ahaṃkāra（梵） | 「我」——那個攥東西的主體本身。 |
| 空 | śūnyatā（梵）／ suññatā（巴） | 不是「沒有」，是「沒有一個攥得住、鐵打的自己」。 |
| 假名有 | prajñapti-sat（梵） | 真真實實地在、能用，可它的「有」是因緣湊成的，不是自個兒就在。 |
| 三輪／三輪體空 | trimaṇḍala / trimaṇḍala-pariśuddhi（梵） | 施者、受者、施物三者；三者都無固定自性、都鬆透了（論中又作「三事不可得」「三種清淨」）。 |
| 般若（波羅蜜） | prajñā / prajñā-pāramitā（梵） | 看透實相、照見空性的智慧——讓施成為波羅蜜的那一點。 |
| 無住（應無所住） | （apratiṣṭhita，無所住） | 心不停、不黏在任何一個相上——《金剛經》給「三輪體空」的另一個名字。 |

## 五、波羅蜜，與三福業

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 波羅蜜 | pāramitā（梵）／ pāramī（巴） | 到彼岸；其義不是「做到完美」，是「無有窮盡」的條件（一切物、一切時、一切種）。 |
| 檀波羅蜜（布施波羅蜜） | dāna-pāramitā | 能把人渡到彼岸去的布施——加了般若、三輪鬆透的施。 |
| 三福業事 | puññakiriyavatthu（巴） | 做福德的三件根本事：施、戒、修。 |
| ——施所成 | dānamaya | 由布施構成的福。 |
| ——戒所成 | sīlamaya | 由持戒構成的福（漢譯亦作「平等業」——平等待眾）。 |
| ——修所成 | bhāvanāmaya | 由修心構成的福。 |
| 戒 | sīla（巴）／ śīla（梵） | 行為的自我整飭；守一條戒，本身也是一種給。 |

## 六、修證的階位與架構

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 預流 | sotāpanna（巴）／ srotaāpanna（梵） | 已踏進解脫之流、回不去的人。 |
| 預流支 | sotāpattiyaṅga | 入流者的記號（前三：對佛法僧的不壞淨；第四：戒、或捨、或慧）。 |
| 不壞淨 | aveccappasāda（巴） | 對三寶不會壞的確信。 |
| 福田 | puññakkhetta（巴）／ puṇyakṣetra（梵） | 能讓供養生出益處的對象——如三寶、有德的人。 |
| 四方僧伽 | cātuddisa saṅgha（巴） | 東南西北、古往今來、一切出家人的整體（含未來、還沒出生的）——供養落於此，誰也據不為己有。 |
| 喜 | pīti（巴）／ prīti（梵） | 修行中生起的歡喜；真喜會往下沉、化成「輕安」。 |
| 輕安 | passaddhi（巴）／ praśrabdhi（梵） | 身心的安息、沉澱（古譯「猗息」「除覺分」）。真給之後，心會安下來；交易之給，只剩躁。 |
| 七覺支 | bojjhaṅga（巴）／ bodhyaṅga（梵） | 通向覺悟的七個因子；喜→輕安的過渡，是分辨真修偽修的試金石。 |
| 思 | cetanā（巴／梵） | 心裡那一念的意向——業的種子。布施以思為首。 |
| 因緣／因 | hetu-pratyaya / hetu（梵） | 生發的親因。布施的因，是你那一念思。 |
| 增上緣 | adhipati-pratyaya（梵） | 幫襯的條件（如眼、手、車馬，乃至今天的手機平台）——是緣，不是因。 |
| 次第說法 | anupubbi-kathā（巴） | 佛對新學者的固定次第：施論、戒論、生天論……布施永遠是第一步。 |
