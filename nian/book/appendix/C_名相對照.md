# 附錄 C　名相對照

正文裡，凡遇一個名相，我都在它頭一次出現時，順手用白話解過了；那些梵文、巴利文的拼音、那一個個帶小點小橫的字母（變音符號），則一概收在這裡，免得擾了正文。

這個附錄，是給想再查對一下的讀者用的。每一條，給出：**中文名相 — 羅馬拼音（巴/梵）— 一句白話**。標「巴」的是巴利文（多用於早期阿含的脈絡），標「梵」的是梵文（多用於大乘的脈絡）；有些詞兩者都給。

---

## 一、念與相關修法

- **隨念** — anussati（巴）／anusmṛti（梵）—— 把佛的功德，記在心裡、不忘地反覆憶念。本書主角；與「正念」是兩條不同的軌道。
- **念佛／佛隨念** — buddhānussati（巴）／buddhānusmṛti（梵）—— 對佛的隨念。本書一路追蹤的，就是這個「念」字的長大。
- **正念** — sati（巴）／smṛti（梵）—— 對當下身心的覺察。與「隨念」同根而別用。
- **九德號** — itipi so（巴，此句的起首語）—— 佛的九種功德名號，是念佛的九道門（見下「九德號細目」）。
- **六隨念** — cha anussati（巴）—— 念佛、念法、念僧、念戒、念施、念天，六個對境，一個引擎。
- **一行三昧** — ekavyūha-samādhi（梵）—— 繫心一佛、專稱名字、念念相續的三昧；禪與淨同出此源。
- **般舟三昧** — pratyutpanna-samādhi（梵，全稱「現前諸佛悉立在前三昧」）—— 於定中親見十方現在諸佛立於眼前。
- **念道** — （漢地早期譯詞）—— 最早期漢譯（《四十二章經》）裡「念佛」尚未成詞時，承擔同一功能的詞。

## 二、定境與三昧

- **三昧** — samādhi（梵／巴）—— 心專注於一境、安定不動的狀態。
- **禪那** — jhāna（巴）／dhyāna（梵）—— 深定；「禪」字即由此來。
- **近行定** — upacāra-samādhi（巴）—— 接近安止、立在定的「門前」的定。
- **安止定** — appanā-samādhi（巴）—— 完全安住、入了定「門裡」的定。

## 三、佛、國、願

- **阿彌陀佛** — Amitābha（梵，無量光）／Amitāyus（無量壽）—— 極樂世界的教主。
- **阿閦佛** — Akṣobhya（梵，不動）—— 妙喜世界的教主；與阿彌陀並列的早期佛國一線。
- **毘盧遮那** — Vairocana（梵，光明遍照）—— 華嚴的本師、法身佛。
- **法藏（菩薩）** — Dharmākara（梵）—— 阿彌陀佛在因地時的名字，發四十八願者。
- **佛國／佛剎** — buddha-kṣetra（梵）—— 佛所成就、教化的國土。
- **淨土** — （對 buddha-kṣetra 清淨面的漢譯）—— 清淨的佛國。
- **極樂** — Sukhāvatī（梵，安樂）—— 阿彌陀佛的佛國名。
- **本願** — pūrva-praṇidhāna（梵）—— 佛在因地所發、成就佛國以救度眾生的根本誓願。
- **法界** — dharmadhātu（梵）—— 一切法的總體；華嚴念佛所通向的、重重無盡的整體。

## 四、心識與本體

- **如來藏** — tathāgatagarbha（梵）—— 眾生心中本具、被煩惱蓋著的清淨佛性。
- **阿賴耶識／藏識** — ālaya-vijñāna（梵）—— 最深層的「倉庫」識，含藏一切業習種子；與如來藏同體異名。
- **末那（識）** — manas（梵）—— 第七識；一刻不停地把藏識執取為「我」的那個作用。
- **轉依** — āśraya-parāvṛtti（梵）—— 把所依的根本（藏識），從染翻轉成淨。念佛被《密嚴經》讀為此翻轉的機制（阿賴耶轉依，ālaya-parāvṛtti）。
- **法身／報身／化身** — dharmakāya／saṃbhogakāya／nirmāṇakāya（梵）—— 佛的三身：常住的體性、酬願所成的莊嚴身、應機示現的變化身。

## 五、行門、緣、果

- **菩提心** — bodhicitta（梵）—— 為利益一切眾生而求覺悟的心。
- **因緣** — hetu-pratyaya（梵）—— 直接生出果的那個正因。
- **增上緣** — adhipati-pratyaya（梵）—— 從旁幫襯、托成的助緣。佛的本願力，於念佛是增上緣（「他力」），非因緣。
- **迴向** — pariṇāmana（梵）—— 把所修功德的方向，整個傾向、轉向給一切眾生與佛道。
- **隨喜** — anumodanā（巴／梵）—— 隨著他人的善而生歡喜，因而分得其福。
- **懺悔（懺摩）** — kṣamā（梵）—— 「懺」即 kṣamā 的音譯「懺摩」；認過去之過、誓往後不造。
- **業障** — karmāvaraṇa（梵）—— 過去惡業所成、遮蔽修行的障礙。
- **無生法忍** — anutpattika-dharma-kṣānti（梵）—— 安住於「諸法不生」的甚深智慧；大乘的高位果證。
- **正定聚／正性決定** — sammatta-niyāma（巴）—— 決定不退、必至涅槃的階位。第四部第十一願所立。
- **凡夫** — pṛthagjana（梵）—— 尚未解脫的平常人；淨土一系所立的救度主體。
- **南無** — namo／namaḥ（梵，歸命、敬禮）—— 「南無阿彌陀佛」的「南無」，是把整個身心，歸投、頂禮過去。

## 六、九德號細目

佛之九德（itipi so 九句），本書第一部第二章詳釋；此處列其巴利原語對照：

- **應供** — arahaṃ（巴）—— 已斷一切煩惱，值得一切供養。
- **正遍知（正等覺）** — sammāsambuddho（巴）—— 圓滿、無遺漏地覺悟一切法。
- **明行足** — vijjācaraṇasampanno（巴）—— 智慧（明）與德行（行）兩皆具足。
- **善逝** — sugato（巴）—— 已善巧地走向涅槃，不再退轉。
- **世間解** — lokavidū（巴）—— 遍知世間的真相。
- **無上士調御丈夫** — anuttaro purisadammasārathi（巴）—— 無上者，能調御一切可度的眾生。
- **天人師** — satthā devamanussānaṃ（巴）—— 天與人的導師。
- **佛** — buddho（巴）—— 覺者。
- **世尊** — bhagavā（巴）—— 福德與智慧具足、世所共尊。

（漢傳「如來十號」的列法，是把上列第六句「無上士調御丈夫」拆為二，並冠以「如來」立首——數目作十，名相與義，與此九句相同。）

## 七、經中人物

- **善財（童子）** — Sudhana（梵）—— 華嚴〈入法界品〉中，南行參訪五十三位善知識、終於普賢的求道者。
- **普賢（菩薩）** — Samantabhadra（梵）—— 主「行」的大菩薩，十大願王的發起者。
- **文殊（師利菩薩）** — Mañjuśrī（梵）—— 主「智」的大菩薩；一行三昧的說者。
- **觀世音（菩薩）** — Avalokiteśvara（梵）—— 法華〈普門品〉中，聞聲救苦、普門示現者。
- **大勢至（菩薩）** — Mahāsthāmaprāpta（梵）—— 楞嚴〈念佛圓通章〉中，以「都攝六根、淨念相繼」念佛者；阿彌陀佛右脅侍。
- **彌勒（菩薩）** — Maitreya（梵）—— 當來下生的未來佛。
- **韋提希（夫人）** — Vaidehī（梵）—— 《觀無量壽經》中，於牢獄請佛說十六觀的王后。
- **摩訶男** — Mahānāma（巴）—— 早期阿含中，向佛請問日常如何安心而引出六隨念的在家弟子。
- **阿難** — Ānanda（梵／巴）—— 佛的侍者；遮婆羅塔「未請佛住世」一事的當事人。

*說明：羅馬轉寫從學界通行式。同一名相若巴利、梵文並存而本書脈絡跨早期與大乘，則並列二者；專屬一系者只列一語。變音符號（如 ā、ṃ、ñ、ṛ、ṭ）為完整轉寫所需，正文一律從簡，完整式集於此。*
