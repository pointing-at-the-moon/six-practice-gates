# 附錄 C　名相對照

正文裡，凡遇梵文、巴利的名相，多半只在首次出現時，用白話夾注一句，之後便直接用漢語詞，不再標注那些帶符號的羅馬字。完整的羅馬轉寫，連同巴利（巴）與梵文（梵）的對照，都收在這裡。

想知道某個詞的原文怎麼寫、它本來是什麼意思的讀者，可以查這張表。不想理會這些的讀者，跳過它，一點不影響讀這本書——這正是把它收在書後的緣故。

（標「巴」者為巴利，「梵」者為梵文；同一詞兩傳統寫法不同者並列。）

## 一、三個骨幹詞

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 懺（懺摩） | kṣamā（梵）／ khama（巴） | 本義近於「容、忍、請人容受」——它求的是被人接住。「懺悔」之「懺」即此字的省稱。 |
| 發露 | deśanā（梵／巴） | 把自己犯的過失，在人面前明明白白說出來。本書真正的主角——「把藏起來的攤到光下」。 |
| 悔（惡作） | kukkucca（巴）／ kaukṛtya（梵） | 心裡那股追悔、坐立難安之情。一個字雙價——有出口則為門，無出口則為牢。 |
| 覆藏 | （paṭicchanna，巴，所覆藏者） | 把過失藏起來、遮蓋住。發露的反面；律藏替它逐日標價。 |

## 二、悔的兩條去路

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 追悔／如法追悔 | kukkucca | 走「見罪→發露→護不更作」這條路的悔——善刃，做完便自行了結。 |
| 掉悔（掉舉與悔） | uddhacca-kukkucca（巴） | 與心的躁動相縛、轉個不停的悔——五蓋之第五，障道之蓋（病刃）。 |
| 五蓋 | pañca nīvaraṇā（巴） | 遮蔽修行的五重障；掉悔是其第五，悔本身不單獨成蓋。 |
| 護不更作 | āyatiṁ saṁvara（巴，未來防護） | 往後守住、不再犯——善悔的封口，向前之力。 |

## 三、僧團的復淨機制

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 僧殘（僧伽婆尸沙） | saṅghādisesa（巴） | 「僧團尚可救治、未至永擯」的一類罪——人雖重傷，尚有殘餘。 |
| 波羅夷 | pārājika（巴） | 最重的一等罪，犯則永失比丘身分，僧團救不回。 |
| 波羅提提舍尼 | pāṭidesanīya（巴） | 「應向一比丘發露便清淨」的一類過——字面即「應發露者」。 |
| 別住（波利婆沙） | parivāsa（巴） | 與清淨大眾分開居住、身分降一格的階段；唯為「覆藏」而設。 |
| 隨覆藏日 | （四分律用語） | 從所量的尺度命名：藏了幾日，別住幾日。與「別住」同指一事。 |
| 本日治 | mūlāya paṭikassanā（巴，送回根本） | 別住期間又犯，已走的日數一筆勾銷、從第一天重來。 |
| 摩那埵 | mānatta（巴） | 別住之後固定六夜的調伏期（不隨覆藏日伸縮）。注疏訓作「悅眾意」「折伏慢心」，屬注疏層。 |
| 出罪 | abbhāna（巴，引出、召回） | 最後一步，把人「還如本位」、重認證為清淨——全律唯一須二十人僧的羯磨。 |
| 羯磨 | kamma（巴）／ karma（梵） | 僧團依法的議決程序；漢譯作「作法」（一白三羯磨：白四羯磨）。 |
| 三斷夜 | ratticchedā（巴） | 使別住夜數中斷、前功不計的三種情形（同宿／別住獨離／不告白）。 |
| 告白 | ārocana（巴） | 行別住者須向來往比丘、於布薩自恣日公開告白其身分——覆藏的結構性反面。 |

## 四、反覆藏的制度

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 布薩（說戒） | uposatha（巴） | 僧團半月一次集合誦戒；前須清淨——帶罪未露者不得說戒、聞戒。 |
| 自恣 | pavāraṇā（巴，恣他舉罪） | 年一度、夏安居竟，每人當眾請僧團就「見、聞、疑」三事舉發己過。 |
| 見、聞、疑 | diṭṭha / suta / parisaṅkā（巴） | 自恣所迎請指摘的三種範圍——遠寬於法庭式的證據門檻。 |
| 瘂法 | （啞法・外道默法） | 相約整季不語的安居——被佛斥為「如怨家共住」，覆藏的集體形式。 |
| 八關齋戒 | aṭṭhaṅga-sīla（巴，聖八支齋） | 在家眾於月六齋日受持的八戒——在家的週期清淨，非「自恣」。 |

## 五、七支與五悔

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 七支供（七支行願） | saptāṅga-pūjā（梵） | 大乘禮拜的七個環節：禮敬、稱讚、供養、懺悔、隨喜、勸請（請住）、迴向。懺是其中唯一的減法之支。 |
| 五悔 | （懺悔・勸請・隨喜・迴向・發願） | 漢傳懺法的核心結構，收於七支懺悔支中——以懺啟端、以願封末。 |
| 三聚 | triskandhaka（梵） | 《大乘集菩薩學論》之範疇：懺悔、勸請、隨喜（不含迴向供養）。《大智度論》作「三事」。 |
| 懺悔業障 | （普賢十大願第四願） | 由七支裡的一「支」，升為十大願裡的一「願」——載入「四喻無盡＋念念相續」的願形。 |
| 四喻無盡 | （虛空界・眾生界・眾生業・眾生煩惱盡） | 一個邏輯構造：設一永不成立的終止條件，故懺「無有窮盡」。十大願共用之底盤。 |
| 念念相續，無有疲厭 | （普賢行願語） | 憑「普賢觀行之力」而非意志苦撐——故無盡而不疲，是願力的綿延，非刑罰的綿延。 |
| 三十五佛懺 | （《決定毘尼經》《優波離會》） | 懺者於三十五佛名號前晝夜獨處至心懺悔——「同一發露，異其證人」的具體化。 |

## 六、三種懺與見好相

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 作法懺 | （智顗判，扶戒律） | 依僧團羯磨程序滅罪——最貼著事相、標準外在可驗。三懺之第一級。 |
| 取相懺（觀相懺） | （智顗判，扶定法） | 於靜心中見罪滅之徵相（見好相）為期——收進定心，事懺長成理懺的脊椎。第二級。智顗原文作「觀相」，後世通稱「取相」。 |
| 無生懺（觀無生懺） | （智顗判，扶慧法） | 觀罪性本空、罪無自相——三懺最高一級，「於懺悔中最尊最妙」。對應《心經》之「空」。 |
| 見好相 | （佛來摩頂・見光見華） | 重罪之懺須見的瑞相，作罪滅之徵驗、戒體可受之開關；相是判準，非歸宿，不可取著。 |
| 罪三品 | （違無作起障道罪／體性罪／無明根本罪） | 三懺各破一品，由淺入深；無生懺「究竟除罪源本」。 |
| 戒定慧（三無漏學） | sīla / samādhi / paññā（巴） | 三條修行主軸；三種懺一一扶之，故成由事入理的階梯。 |

## 七、無生與空

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 無生 | anutpāda（梵，不生） | 罪本不曾以自性的方式生起；即《心經》「不生不滅」落到罪業上。 |
| 罪性本空 | （罪性不在內外中間） | 罪沒有一個離開眾緣而自存的實性——尋之悉虛。非謂無因果。 |
| 空 | śūnyatā（梵）／ suññatā（巴） | 不是「沒有」，是「沒有一個鐵打的自性」。懺門所對的《心經》之字。 |
| 實相 | （諸法實相・第一義空） | 諸法本來的樣子；「端坐念實相」即懺之最深一照。 |
| 無罪相懺悔 | （《觀普賢觀經》自題） | 懺到照見罪「無自相」的地步——被否定的是「罪相」，不是「罪」（曾造業的事實）。 |
| 慧日／霜露 | （眾罪如霜露慧日能消除） | 罪之消是慧日「照見」（認知之轉），非赦免、非情緒紓解；霜露本非實有，故日照即消。 |
| 我執／我所 | ahaṃkāra / mamakāra（梵） | 「我」與「我的」之執；「失我」所失者即此（能懺之我），非所懺之罪。 |
| 封惑自縛 | （竺道生語） | 罪相之牢固，是人親手把迷惑封存成實有再自縛；照見性空，繩結自鬆。 |
| 業果不空 | （空不撥因果） | 罪性雖空，業之果報不空——「不得罪，不得不罪」雙遮。全卷最高守點。 |

## 八、日用相續

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 六時懺（晝夜六時） | （晝三時夜三時） | 一日六次面對諸佛行懺——把懺從「遇事而起」改寫成「按時而行」的日課。 |
| 三業 | （身・口・意三業） | 染污由身口意而起，清淨也從這同一道門反向而入。 |
| 五逆／無間業 | pañca ānantarika（梵） | 五種最重之罪（弒父弒母等）。「五逆」阿含已有；「無間業」為後世阿毘達磨系統化之名相。 |
| 無根之信 | （阿闍世所得） | 「信而無根」——弒父斷了證果之根，所餘失了道根支撐、卻仍真實的勝解之信。 |
| 五分法身 | （戒・定・慧・解脫・解脫知見） | 受新戒體所成就的「重生之身」——死而後生之「生」的經文自名。 |
