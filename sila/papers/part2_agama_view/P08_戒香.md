## 摘要

本卷第二部前三篇已立八聖道之三戒支（P05）、五戒之經典範圍跨戒差別（P06）、八關齋戒之週期性出家機制（P07）；然週期出家層畢竟為例外性高擡，在家眾日常持五戒於社會場域之顯現如何發生，為本篇所欲展開之經典提問。本文之核心主張為：**戒香（*sīla-gandha*）非單為文學譬喻，乃戒之社會場域效應於經文層、論藏層、律宗註疏層三軸交織之經典教義別名**。經文層證據為《雜阿含》卷三十八經1073《戒香經》、《別譯雜阿含》卷一經十二、《增壹阿含》卷十三香品三漢譯本跨二百五十年之架構穩定；論藏層證據為玄奘譯《瑜伽師地論》卷二十二攝決擇分聲聞地之戒蘊三相（無失壞 / 自性 / 自性功德）與六異門列舉（戒香 / 戒塗香 / 戒嚴具 等）之官方教義編定；律宗註疏層證據為道宣《四分律行事鈔》卷下引《華嚴》三香（戒香 / 定香 / 解脫香）之階梯映射至五法蘊。《增支部》第十一集第一經之十一支鏈條進一步揭示：戒香之社會場域可感知效應非靜態屬性，乃「漸次至最上」之發展性次第之在家側入口——自此進入五法蘊之經典分層修行體系。

## 一、問題意識

P07 已立八關齋戒（*aṭṭhaṅga-uposatha*）之週期性出家機制——在家眾於布薩日（*uposatha*）借取出家戒禁忌之週期出家層，以一日一夜（*ahorātrika-saṃvara*，近住律儀）之經典設計性時限，效法阿羅漢（「我效阿羅漢」，*arahataṁ anukaromi*）之住處。然此週期出家層之儀軌結構畢竟為例外性高擡，非在家眾日常持戒之經典代表性表述。在家眾於日常持五戒（殺、盜、邪婬、妄、酒）之基底，即令未受持週期出家之八關齋，其戒之社會場域顯現如何發生？即令未進入週期出家層之儀軌，僅持五戒之在家側基底，其持戒清淨之效應是否亦於社會場域中可感知？此一提問之答案於《雜阿含》第1073經（對應《增支部》3.79 *Gandha-sutta* 之漢譯）中得經典內生表述。[^1]

經文中，尊者阿難獨一靜處，作如是思惟：「有三種香，順風而熏，不能逆風。何等為三？謂根香、莖香、華香。或復有香，順風熏，亦逆風熏，亦順風逆風熏耶？」佛陀肯定其問，並答曰：「有善男子、善女人，在所城邑、聚落，成就真實法，盡形壽不殺生、不偷盜、不邪婬、不妄語、不飲酒。如是善男子、善女人，八方上下，崇善士夫，無不稱歎言：『某方某聚落善男子、善女人，持戒清淨，成真實法』。」並以偈頌收束：「非根莖華香，能逆風而熏；唯有善士女，持戒清淨香。逆順滿諸方，無不普聞知。」[^3] 此一「善士女持戒清淨香」（字義「善士之香」，*sappurisa-gandha*）之句式，非比丘僧團對在家眾之儀禮性期許，乃經文自身之架構性句法，明確標出「持戒清淨之社會場域可感知效應跨越自然香氣之物理限制」之經典內生表述。

戒香跨越自然香氣物理限制之讀法，並非單為譬喻層之文學修辭。於玄奘譯《瑜伽師地論》卷二十二攝決擇分聲聞地中，世尊明列「戒蘊略義，有三種相：一者、無失壞相，二者、自性相，三者、自性功德相」，並於六異門（梵文 *ṣaṭ-paryāya*，即戒之六種官方教義別名）中明列「戒薰香」（*sīla-vāsana*）、「戒塗香」（*sīla-vilepana*）兩詞為戒之官方別名：「有處世尊宣說尸羅名為薰香。如伽他說：阿難！有香類，順風善能薰；逆風亦能薰。順、逆薰亦爾。」並答曰：「具戒士夫補特伽羅，遍諸方域，妙善稱譽聲頌普聞……是故尸羅名為薰香。」[^12] 戒香於經文層為譬喻，於論藏層升為教義別名——成為「具戒士夫補特伽羅遍諸方域妙善稱譽聲頌普聞」之社會場域可感知架構表述之官方名相。

此一架構讀法進一步由跨譯本之穩定所證實。漢譯側具三譯本之互證：T0099《雜阿含》卷三十八經1073《戒香經》（求那跋陀羅譯，劉宋元嘉435–443）、T0100《別譯雜阿含》卷一經十二《香經》（失譯，約四至五世紀）、T0125《增壹阿含》卷十三香品（瞿曇僧伽提婆譯，東晉四至五世紀）。三本所載戒香教義之核心句式——「三香順風與戒香雙向」之分類、「戒香最為上」（或「戒香最為勝」）之優位句、「逆順滿諸方」（或「十方悉聞之」、「逆順悉聞之」）之社會場域廣傳句——於三譯本中無刪除、無增添、無倒置之核心架構句式於跨譯本之穩定。此一跨譯本架構穩定，為戒香教義之非衍生性、經典結構性教義之語文學證據。[^2]

本篇之核心主張即：**戒香教義之架構機制非單為經文層譬喻之文學修辭，乃戒之社會場域效應於經文層（《增支部》3.79 + 漢譯三譯本）、論藏層（《瑜伽師地論》卷二十二戒蘊三相 + 六異門）、律宗註疏層（道宣《四分律行事鈔》引《華嚴》三香）三軸交織之經典教義別名**。《增支部》第十一集第一經（*Kimatthiyasutta*）之十一支鏈條進一步揭示：戒香之社會場域可感知效應非靜態屬性，乃「漸次至最上」（*anupubbena aggāya parentī*）之發展性次第之入口——戒香為在家側可感知之表層，自此進入五法蘊（*pañca-dhamma-kkhandha*，即戒蘊 / 定蘊 / 慧蘊 / 解脫蘊 / 解脫知見蘊；非五取蘊）之經典分層修行體系。

## 二、經典依據

### 2.1 巴利軸——巴利原文之檔案庫處境與十一支鏈條之獨立性

本文之巴利錨點具一獨特處境須誠實揭露。《增支部》3.79 *Gandha-sutta*（巴利學會編號 PTS:AN i ~225，屬增支部三集 Ānanda-vagga）雖於南傳上座部巴利傳統中為穩定之經典片段，然作者所用之巴利原文資料庫其中所收二十七種《增支部》經典中，並未收 AN 3.79 與 AN 3.80 兩經之巴利原文。[^2] 同樣，《法句經》第54至56偈（Dhp 54-56，屬 Puppha-vagga 花品，巴利傳統載戒香譬喻之第二處）之巴利原文亦不於本文之資料庫中。[^6] 又《長部》之沙門果經（DN 2）、合誦經（DN 33）、十上經（DN 34）、《中部》之諸經（MN 27、38、51、107）、《如是語》51經、《清淨道論》第一章正文等九處重要巴利錨點，於本文之巴利資料庫中皆缺位。[^9] 故本文之巴利層引用，以巴利學會編號（PTS）為標，不作整段原文之逐字引；漢譯三譯本為本篇之主要文本錨。

雖巴利原文之直接逐字引不可得，然 AN 3.79 戒香譬喻之巴利核心結構為當代巴利學界（Bhikkhu Bodhi 2012 之《增支部》全譯本、Bhikkhu Sujato 在 SuttaCentral 之 CC0 公開譯本）反覆引述、討論之穩定文本。漢譯三譯本與巴利傳統之意象於核心架構句法無歧異；此一誠實揭露本身為當代數位文獻學紀律之要求，不影響本文跨譯本架構穩定主張之實質。又須慎防一誤判：戒香之意象並非漢傳大乘之後典化綜合——其源於 AN 3.79 巴利傳統與《法句經》Puppha-vagga 之雙重經典基底，並經玄奘譯《瑜伽師地論》卷二十二之梵文直譯交叉驗證（見 §五.1）。

獨立於 AN 3.79 之缺位，本文之另一巴利主錨為《增支部》第十一集第一經 *Kimatthiyasutta*（中譯《何義經》，漢譯側對應《中阿含》卷十經第四十二《何義經》第一，瞿曇僧伽提婆譯）。此經之巴利原文於本文資料庫中具完整錄入（Bhikkhu Sujato CC0 譯本）：

> *kimatthiyāni, bhante, kusalāni sīlāni kimānisaṁsānī"ti?*
> *"Avippaṭisāratthāni kho, ānanda, kusalāni sīlāni avippaṭisārānisaṁsānī"ti.*
> *……*
> *Iti kho, ānanda, kusalāni sīlāni anupubbena aggāya parentī"ti.*

漢譯（義譯）：「世尊！持戒以何義？何利？」「阿難！持戒以無悔為義、為利……如是，阿難！持戒漸次而至最上。」[^7]

此經之十一支鏈條——*sīla → avippaṭisāra → pāmojja → pīti → passaddhi → sukha → samādhi → yathābhūta-ñāṇadassana → nibbidā → virāga → vimutti-ñāṇadassana*（戒 → 無悔 → 歡悅 → 喜 → 輕安 → 樂 → 定 → 如實知見 → 厭離 → 離欲 → 解脫知見）——為戒之發展性次第之巴利原典證據，於本文 §三.3 之架構讀法中為核心錨點。另《增支部》第五集第三十二經 *Cundīsutta* 之預流支軸（*ariyakanta-sīla*，聖者所愛之戒）為次階補錨，本篇引為旁證（本系列 P02 沿用）。[^8]

### 2.2 漢譯三譯本——跨二百五十年之架構穩定

漢譯之主要文本錨為《雜阿含》卷三十八經1073《戒香經》（求那跋陀羅譯，劉宋元嘉435–443，屬說一切有部所傳）。經文之開卷敘事與五戒列舉已於 §一引出，其偈頌十二句之完整版本為：

> 「非根莖華香，能逆風而熏，唯有善士女，持戒清淨香。
>  逆順滿諸方，無不普聞知，多迦羅栴檀，優鉢羅末利。
>  如是比諸香，戒香最為上，栴檀等諸香，所熏少分限。
>  唯有戒德香，流熏上昇天，斯等淨戒香，不放逸正受。
>  正智等解脫，魔道莫能入，是名安隱道，是道則清淨。
>  正向妙禪定，斷諸魔結縛。」[^3]

此偈頌之最末四句——「是名安隱道，是道則清淨。正向妙禪定，斷諸魔結縛」——將戒香從「社會場域之可感知顯現」推進至「定（*samādhi*）之導向」與「魔結之斷除」，為戒香教義內生之「戒至定」之鏈條伏筆（於本文 §三.3 + §六之向 P24 之伏線中再述）。

漢譯之第二譯本為《增壹阿含》卷十三香品（瞿曇僧伽提婆譯，東晉四至五世紀，屬大眾部相關所傳）：

> 世尊告曰：「此三種香，亦逆風香，亦順風香，亦逆順風香。」阿難言：「何等為三？」世尊告曰：「戒香、聞香、施香。是謂，阿難！有此香種，然復此逆風香，亦順風香，亦逆順風香。諸世間所有之香，此三種香最勝、最上，無與等者，無能及者。……」
>
> 「木蜜及栴檀，優鉢及諸香，亦諸種種香，戒香最為勝。
>  此戒以成就，無欲無所染，等智而解脫，逝處魔不知。
>  此香雖為妙，及諸檀蜜香，戒香之為妙，十方悉聞之。」[^4]

T0125 之三香分類列「戒香、聞香、施香」——三皆「逆風、順風、逆順風」之雙向；戒香於此三中為「最勝、最上」。此一三香分類為本文 §三.2 之三香階層讀法之經文層證據。

漢譯之第三譯本為《別譯雜阿含》卷一經十二《香經》（失譯，約四至五世紀）。其偈頌簡要：

> 「持戒香丈夫，芳馨遍世界，名聞滿十方，逆順悉聞之。」[^5]

T0100 雖文字較簡，然其「逆順悉聞之」之社會場域廣傳句與 T0099 之「逆順滿諸方，無不普聞知」、T0125 之「十方悉聞之」於核心句法上穩定一致。三譯本所載戒香教義之核心句式——「三香順風與戒香雙向」之分類、「戒香最為上」之優位句、「逆順 / 十方 / 滿諸方」之廣傳句——於跨二百五十年之三譯場（劉宋 / 東晉 / 失譯）中無刪除、無增添、無倒置。此一跨譯本架構穩定，為戒香教義為「經典結構性教義」（非「在家側牧靈譬喻」、非「漢傳後典化綜合」）之語文學證據。

### 2.3 漢譯側其他旁證——《長阿含》沙門果經敘事框架與三蘊五蘊列舉

漢譯側之第三層旁證為《長阿含》卷十七沙門果經（佛陀耶舍 + 竺佛念合譯，弘始412–413，屬法藏部）。沙門果經之巴利對應為 DN 2，以阿闍世王（Ajātasattu）向佛陀請問沙門之果為主敘事框架；經中世尊以戒蘊（*sīla-kkhandha*）之三層列舉（小戒 / 中戒 / 大戒，*cūḷasīla / majjhimasīla / mahāsīla*）為應答之第一層。然本文所據之資料庫中，T0001 卷十七雖含阿闍世王敘事框架，然戒蘊三層列舉之完整段落不於資料庫中。[^10] 故本文於 T0001 卷十七僅以敘事框架旁證引，戒蘊三相之主錨用《瑜伽師地論》卷二十二（見 §五.1）。又《長阿含》卷八眾集經（對應 DN 33 *Saṅgītisutta*）、卷九十上經（對應 DN 34 *Dasuttarasutta*）中亦見三蘊（戒 / 定 / 慧）與五法蘊（戒 / 定 / 慧 / 解脫 / 解脫知見）之漢譯側列舉，本文引為旁證。[^11]

## 三、戒香之經文層架構讀法

### 3.1 三香順風與戒香雙向——《雜阿含》戒香經之四重句法

T0099 戒香經之經文層架構，可析為四重句法。

**第一重**為阿難之開卷問句：「有三種香，順風而熏，不能逆風。何等為三？謂根香、莖香、華香。或復有香，順風熏，亦逆風熏，亦順風逆風熏耶？」此一問句之架構性結構，並非對自然香氣物理現象之單純好奇，而係阿難於佛陀座前以「假設性反問」之句式預設一種跨越自然限制之香類之可能。阿難並非疑問「是否有逆風熏之香」，而係邀請佛陀肯認「有逆風熏之香」並指明其名相。問句之語氣本身已暗含答案：此種香確有，但須佛陀為其揭示。

**第二重**為佛陀之肯定回答：「如是！如是！有三種香，順風熏，不能逆風……阿難！亦有香，順風熏、逆風熏、順風逆風熏。」此一三疊「順風熏、逆風熏、順風逆風熏」之雙向加雙向之並列句法，於漢語語法中之累積效應，為跨越自然順風物理限制之教義內生表述。雙向不止於「逆風」，更包含「順風逆風」之完整方向集合——即戒香之社會場域廣傳於「滿諸方」、「八方上下」、「十方」之表述基底。

**第三重**為佛陀之社會場域指明：「有善男子、善女人，在所城邑、聚落，成就真實法，盡形壽不殺生、不偷盜、不邪婬、不妄語、不飲酒。如是善男子、善女人，八方上下，崇善士夫，無不稱歎言：『某方某聚落善男子、善女人，持戒清淨，成真實法』。」此一句法之架構性意義為：**戒香之逆風雙向之承載者非「香類」，而係「善士女」之社會場域可感知存在**。香類非物，香即善士女；香之雙向逆順非香氣之物理現象，乃善士女之「稱譽聲頌」（用《瑜伽師地論》卷二十二玄奘譯詞）於社會場域之傳播。

**第四重**為偈頌之收束。偈頌之十二句，自「非根莖華香」之否定句起，以「唯有善士女，持戒清淨香」之肯定句承，以「逆順滿諸方，無不普聞知」之社會場域句拓展，以「戒香最為上」之優位句鎖定，並以「正向妙禪定，斷諸魔結縛」之戒至定鏈條句為向《增支部》11.1 十一支鏈條之內生伏筆收束。此一四重句法非「香之譬喻」之文學修辭，乃「戒之社會場域效應」之經典教義內生表述。

### 3.2 三香分類——《增壹阿含》之戒/聞/施階層與道宣《行事鈔》引《華嚴》之擴展

T0125 增壹阿含 香品所列之「戒香、聞香、施香」三香分類，於漢譯側為戒香教義之第二層架構拓展。三香皆為「逆風、順風、逆順風」之雙向香類；然「戒香最為勝」、「戒香之為妙，十方悉聞之」之優位句，明示三香於相對位序中——戒香為三香之首。

此一三香分類於漢傳律宗註疏層更獲一更高層之三香擴展。道宣《四分律行事鈔》卷下（唐七世紀，屬律宗南山宗）於八關齋戒之布薩儀軌段落中，引《華嚴》之三香偈作為香供養之偈頌：

> 各說偈言，《華嚴》云：「戒香定香解脫香，光明雲臺遍法界，供養十方無量佛，見聞普熏證寂滅。」[^18]

此一三香——戒香 / 定香 / 解脫香——之擴展，將 T0125 之「戒 / 聞 / 施」三香之社會場域層轉換為「戒蘊 / 定蘊 / 解脫蘊」之三蘊修行階梯之經典映射。三蘊復可進一步擴展為五法蘊（*pañca-dhamma-kkhandha*）：戒蘊 / 定蘊 / 慧蘊 / 解脫蘊 / 解脫知見蘊（依《如是語》第51經 + DN 33 + DN 34 之巴利原典列舉，前述以巴利學會編號為參考之巴利錨點）。戒香於五法蘊架構中為入口位——自社會場域之可感知層入，沿三香階梯內入，至五法蘊之深層修行體系。此一架構之內生映射為 §五.5 道宣引《華嚴》三香擴展之經文層基底。

### 3.3 《增支部》11.1 十一支鏈條——戒之發展性次第而非靜態集合

戒香教義之第三層架構讀法，為《增支部》第十一集第一經（*Kimatthiyasutta*）之十一支鏈條表達之「戒蘊為發展性次第而非靜態集合」之主張。此一主張為本系列 SĪLA 卷之主要架構伏筆。

十一支鏈條為：*sīla（戒） → avippaṭisāra（無悔） → pāmojja（歡悅） → pīti（喜） → passaddhi（輕安） → sukha（樂） → samādhi（定） → yathābhūta-ñāṇadassana（如實知見） → nibbidā（厭離） → virāga（離欲） → vimutti-ñāṇadassana（解脫知見）*。經之收束句為 *Iti kho, ānanda, kusalāni sīlāni anupubbena aggāya parentī*（「如是，阿難！持戒漸次而至最上」）。

「漸次至最上」（*anupubbena aggāya parentī*）之表述揭示一架構性主張：**戒蘊非分疇式靜態集合，乃發展性次第之入口**。戒蘊作為五法蘊之首蘊，非與其他四蘊並列之靜態名相，而係沿十一支鏈條漸次而至定、慧、解脫、解脫知見之發展性次第之始發點。此一發展性次第之主張與 T0099 戒香經偈頌末四句之「正向妙禪定，斷諸魔結縛」之戒至定鏈條句之經文層伏筆相互印證：戒香之社會場域可感知層，非戒之終點，乃戒之發展性次第之在家側可感知入口。

戒香之社會場域可感知效應，因此非單純之在家側倫理表述，而係十一支鏈條之在家側入口——即令在家眾未至七支（戒 → 無悔 → 歡悅 → 喜 → 輕安 → 樂 → 定）之止觀深修，其於日常持五戒之社會場域顯現中，已於發展性次第之第一支（戒 → 無悔）中行進。此一「戒 → 無悔」之內生轉化於 T0099 偈頌之「斯等淨戒香，不放逸正受」（「不放逸正受」對應 *appamādo samādhi*，不放逸之正受）中得內生表述。十一支鏈條前七支為持戒至定之自然湧現之架構性序列，為本系列 P24《楞嚴四清淨明誨》之戒至定之經典伏筆所承（於戒至定接續中之在家側預埋）。

### 3.4 酒戒之社會場域可感知與論藏層列位之分流——P06 沿用之 P08 進深

於本文 §一所引 T0099 戒香經之五戒列舉中，「不殺生、不偷盜、不邪婬、不妄語、不飲酒」五戒全數列為戒香之社會場域可感知效應之承載戒，**酒戒於此列舉中與其他四戒並列**。然 P06 已立酒戒於經典 4 業道（殺 / 盜 / 婬 / 妄）與 10 業道（*daśa karmapathāḥ*）之論藏編定中皆 在列舉外——《大乘阿毘達磨集論》（無著造，玄奘譯，T1605）卷四與《大乘阿毘達磨雜集論》（安慧造，玄奘譯，T1606）卷七至卷八之 10 業道列舉明列「殺 / 盜 / 婬 / 妄 / 兩舌 / 惡口 / 綺語 / 貪 / 瞋 / 邪見」十項，飲酒不於 10 業道之內。[^13][^14]

此一論藏層之 10 業道列舉與經文層之戒香列舉之差別，並非教義之矛盾，而係兩層之架構性分流：

**論藏層**(T1605 + T1606 之 10 業道)為「業道」之分疇式列舉——以「業之究竟動機」為分疇判準（殺 = 瞋之究竟、盜與婬 = 貪之究竟、妄 = 三毒隨一之究竟）。酒戒不入 10 業道，係因飲酒本身非「業之究竟動機」，而係「護持道本身」（P06 主場）——飲酒之過患不在於「業之染污」本身，而在於失念而致他四戒之難守。

**經文層**(T0099 戒香經之五戒列舉)為「戒香之社會場域可感知效應」之列舉——以「持戒清淨於社會場域之顯現」為列入判準。於此判準下，酒戒之持守同樣為社會場域可感知：「某方某聚落善男子、善女人，持戒清淨」之社會場域稱譽，自然包含酒戒之不違越。

此一分流之架構性意義為：**酒戒之社會場域可感知效應雖無「業道染污致動」之本體論地位，然其戒香之社會場域可感知層同樣顯現**。「護持道本身」之性格不否定「社會場域可感知」之性格——此為 P06 之經典範圍跨戒差別之 P08 進深（P06 雙重語域於本篇之沿用）。

## 四、學者對話

本系列自 P04 起確立之「讓+推」二步學者對話結構，於本卷 P05、P06、P07 三篇中沿用。本篇為本卷之第八篇，延續同一三家陣容（Anālayo、Gethin、Bodhi），並以 Harvey 為次階學者引用（沿 P06 + P07 之次階紀律）。

### 4.1 Anālayo——跨譯本穩定性之方法論架構性提昇

**讓**:Bhikkhu Anālayo 之《雜阿含研究》（*Saṃyukta-āgama Studies*,Dharma Drum Publishing 法鼓 2015）為跨漢譯與巴利對應之文獻穩定性方法論之當代範式。其對《雜阿含》、《增壹阿含》、《長阿含》與巴利《增支部》、《長部》之跨譯本比較讀法為當代學界之主要方法範例。本篇對 AN 3.79 戒香經之三漢譯本(T0099 + T0100 + T0125)之跨譯本穩定性之應用，即承自 Anālayo 之方法。

**推**:Anālayo 之跨譯本方法，其陳述之終點為「文獻層之文獻穩定性」——亦即，證明三譯本之核心句法穩定為「同源經典之可靠傳承」之證據。本篇於此推進一步——**跨三譯本之二百五十年之架構穩定，不止於同源傳承之證據，更係「戒香教義為經典結構性教義（非在家側牧靈式譬喻、非漢傳大乘後典化綜合）」之語文學證據**。文獻層穩定為「傳承可靠」之證據，架構層穩定為「教義性質」之證據——後者為前者於教義性格上之進階推論。此一進階推論於本文 §五.1 之《瑜伽師地論》卷二十二六異門之論藏層編定中獲跨層之相互印證。

### 4.2 Gethin——三蘊 / 五蘊列舉之發展性次第讀法

**讓**:Rupert Gethin 之《佛教基礎》（*The Foundations of Buddhism*,Oxford University Press 1998）第八章對戒 / 定 / 慧三蘊（*tisso sikkhā*）之架構性列舉，以及對五法蘊（戒 / 定 / 慧 / 解脫 / 解脫知見）之經典擴展之標準陳述，為當代學界對戒蘊（*sīla-skandha*）之架構性列舉之主要參照。Gethin 第九章「佛教社群」（The Buddhist community）對僧團與在家眾界面之社會場域框架，直接相關於本篇之戒香社會場域效應主題。

**推**:Gethin 之三蘊 / 五蘊列舉，其陳述之終點為「教義性結構列舉」——亦即，將戒 / 定 / 慧三蘊與五法蘊呈現為佛教教義之分層架構之標準陳述。本篇於此推進一步——**《增支部》11.1 之十一支鏈條之「漸次至最上」（*anupubbena aggāya parentī*）表述揭示：五法蘊之戒解脫蘊架構並非靜態分層集合，而係發展性次第之架構性表述**。戒香為發展性次第之在家側可感知入口——非五蘊外之譬喻層附加，乃五蘊內生之發展性次第之第一支於社會場域之可感知顯現。Gethin 之標準列舉為本篇發展性次第讀法之前置基底；發展性次第讀法為標準列舉之動力性深化。

### 4.3 Bodhi——四面和諧框架至論藏編定之架構性深化

**讓**:Bhikkhu Bodhi 之《八聖道》（*The Noble Eightfold Path*,Buddhist Publication Society 1984 / Wisdom Publications 1994）對正語 / 正業 / 正命三戒支之「社會 — 心理 — 業 — 觀」四面和諧之框架，直接相關於本篇主題。Bodhi 之《增支部全譯》（*The Numerical Discourses of the Buddha*,Wisdom Publications 2012）為《增支部》之全譯本，其三集之導論中含 AN 3.79 之譯文與註腳。Bodhi 之《皈依與受戒》（*Going for Refuge and Taking the Precepts*,Buddhist Publication Society Wheel 282/283/284,1981）第二部分為五戒之主題專論（P06 沿用）。

**推**:Bodhi 之四面和諧框架，其陳述之終點為「戒之四面之觀念性整合」——亦即，將戒之社會、心理、業、觀照四維度作為一觀念性整體呈現。本篇於此推進一步——**玄奘譯《瑜伽師地論》卷二十二之戒蘊三相（無失壞 / 自性 / 自性功德）與六異門（戒香 / 戒塗香 / 戒嚴具 等）為四面和諧框架於唯識論藏層之經典官方編定**。Bodhi 之四面和諧為經文層之觀念性整合；《瑜伽師地論》之戒蘊三相加六異門為論藏層之教義性官方編定。後者為前者之經典結構性編定——亦即，將觀念性整合提昇為論藏層之教義別名編定。此一深化為本文 §五.1 之主章內容。

### 4.4 Harvey——次階學者引用

Peter Harvey 之《佛教倫理導論》（*An Introduction to Buddhist Ethics*,Cambridge University Press 2000）第二至第三章對五戒之社會基底之陳述與第四章對僧 / 俗社群界面之處理，為本篇戒香社會場域效應之次階學者引用。Harvey 於本系列 P06 + P07 + P08 中為次階陳述，不入三家陣容，僅於註腳層引用。

### 4.5 西方學界對 AN 3.79 之「稀薄關注」之誠實揭露

於本篇撰寫前之西方學界文獻調查中發現一架構性事實須誠實揭露：**西方學界對 AN 3.79 *Gandha-sutta* 與戒香教義甚少有專論性處理**。Anālayo、Gethin、Bodhi、Harvey 諸家於其主要著作中皆未對 AN 3.79 戒香教義作專章專論；戒香於西方學界文獻中主要以「附帶引用」之地位出現，而非主要學術對象。

此一「稀薄關注」本身為一架構性發現——**戒香教義於印度與漢譯架構之強度，與其於西方接受之稀薄之間，構成一不對稱關係**。AN 3.79 之三漢譯本跨二百五十年之架構穩定、玄奘譯《瑜伽師地論》卷二十二之六異門官方編定、道宣《四分律行事鈔》引《華嚴》三香之律宗註疏層擴展之三軸交織，於印度與漢譯傳統中為一強度甚高之教義交織；而西方學界對此之接受卻僅以零散註腳出現。本篇為當代學術對戒香教義作一專章專論之嘗試，以校正此一不對稱之稀薄關注。

## 五、論藏編定、註疏層整合與律藏旁證

### 5.1 《瑜伽師地論》卷二十二——戒蘊三相與六異門之論藏層官方編定（本篇單一最強教義錨）

本篇之單一最強教義錨為玄奘譯《瑜伽師地論》（*Yogācārabhūmi*,T1579，唐貞觀646–648）卷二十二攝決擇分聲聞地中之戒蘊三相與六異門列舉。此一段落為戒香教義由經文層譬喻提昇為論藏層官方教義別名之關鍵編定。

**戒蘊三相**(*sīla-skandhasya tisro lakṣaṇāḥ*）之列舉為：

> 世尊顯示戒蘊略義，有三種相。一者、無失壞相，二者、自性相，三者、自性功德相。此復云何？謂若說言：「安住具戒」。由此顯示尸羅律儀無失壞相。若復說言：「能善守護別解律儀」。由此顯示尸羅律儀自性相。若復說言：「軌則、所行皆悉圓滿」。由此顯示別解律儀，如其所受，觀他增上自性功德相。[^12]

戒蘊之三相分別對應於受持戒性（受戒之承擔）、出離戒性（別解脫律儀，*pātimokkha-saṃvara* / *prātimokṣa-saṃvara*,P07 沿用）、修習戒性（觀他增上之自性功德）。

**六異門**(*ṣaṭ-paryāya*，即「戒之六種官方教義別名」）中與戒香相關者為：

> 有處世尊宣說尸羅名為塗香。如伽他說：「苾芻、苾芻尼，戒塗香圓滿。於不善能捨；於善能修習。」
>
> 有處世尊宣說尸羅名為薰香。如伽他說：「阿難！有香類，順風善能薰；逆風亦能薰。順、逆薰亦爾。」答：「具戒士夫補特伽羅，遍諸方域，妙善稱譽聲頌普聞。譬如種種根莖香等，隨風飄颺遍諸方所，悅意芬馥，周流彌遠。是故尸羅名為薰香。」[^12]

此處須注意三梵文語域之精確區辨：

- *sīla-gandha*（戒香）為「氣味本身」之語域（*gandha* 之純粹氣味義）；
- *sīla-vilepana*（戒塗香）為「塗於身之膏香」之語域（*vilepana* 之塗抹義）；
- *sīla-vāsana*（戒薰香）為「餘香之薰染」之語域（*vāsana* 之薰染留香義）。

三梵文語域於漢譯側皆作「戒香」或「薰香」之譯，有混同之效；然於梵文原文中，三詞之概念範圍略有差別。本文於述戒香教義時，精確標明三語域之區辨，不以單一「戒香」一詞籠統涵蓋——此一語域之精確區辨為本文之語文學紀律之一，亦避免漢譯側之同詞混同所致之誤讀。

戒蘊三相與六異門之論藏層官方編定，將戒香由經文層之文學譬喻提昇為唯識阿毘達磨層之教義別名地位——亦即，**戒香於唯識論藏層為戒之社會場域效應之官方教義名相，具相當於戒蘊本身之教義位**。此一論藏層編定為本文核心主張之單一最強證據。

### 5.2 《大乘阿毘達磨集論》與《雜集論》之別解脫律儀與十業道列舉

### 5.5 道宣《四分律行事鈔》卷下引《華嚴》三香——律宗註疏層之擴展

道宣（596–667，律宗南山宗祖）《四分律行事鈔》卷下(T1804,唐七世紀)於八關齋戒之布薩儀軌段落中，以《華嚴》三香偈為香供養之偈頌引用（原文已於 §三.2 引出）。此引《華嚴》之三香——戒香 / 定香 / 解脫香——之擴展，於律宗註疏層中具兩重架構性意義：

**第一**為三香階層之擴展：T0125 增壹阿含 香品之三香（戒 / 聞 / 施）為社會場域層之同階列舉，「戒香最為勝」為三香之優位句；而道宣引《華嚴》之三香（戒 / 定 / 解脫）為三蘊修行階梯之階層映射。此一階層映射為 §三.2 三香擴展之律宗註疏層基底。

**第二**為五法蘊之內生映射：戒 / 定 / 解脫三香結合慧（隱含於戒與解脫之間）、解脫知見（隱含於解脫之後），即構成五法蘊（戒蘊 / 定蘊 / 慧蘊 / 解脫蘊 / 解脫知見蘊）之經典映射。道宣引《華嚴》三香之入律宗八關齋儀軌，使戒香教義之社會場域層、三蘊階層、五法蘊修行架構三層映射於律宗儀軌中具體落實。

道宣《行事鈔》之主要內容於本卷 P17（道宣《行事鈔》全展開）為主場保護所限，本篇於 T1804 僅就此引《華嚴》三香之短引處作架構性引用。另《根本說一切有部毘奈耶》（義淨譯，T1442，唐七至八世紀）卷五之「具足尸羅者，謂具戒蘊。有勝善法者，謂具定蘊。言梵行者，謂具慧蘊」之三蘊對應列舉，於律藏內部展示戒 / 定 / 慧三蘊之列舉結構，與《華嚴》三香之擴展遙相呼應（本卷 P09 + P10 主場保護所限，本篇僅作短引）。[^19]

### 5.6 智顗與永明延壽戒香零命中之架構性缺位——天台與律宗之教義分工

於本系列之文獻調查中，本篇進一步確認一架構性缺位之事實：智顗（538–597，天台宗祖）《摩訶止觀》(T1911)、《菩薩戒義疏》(T1812)二著作中，戒香一詞之命中數為零；戒之廣詞（持戒 / 尸羅 / 禁戒 / 戒蘊）於《摩訶止觀》中命中五十一次，然「戒香」單獨之教義別名命中為零。永明延壽（904–975）《萬善同歸集》(T2017)之戒香命中亦為零。[^20]

此一架構性缺位並非智顗對戒香教義無觀念，而係天台與律宗之教義分工：**智顗將戒之觀念上昇至圓頓戒 / 一心戒之般若與中觀層，將社會場域可感知層之戒香教義讓予道宣律宗（南山宗）之註疏體系**。此一教義分工之架構性意義為——同一教義（戒之社會場域效應）於不同宗派之教義體系中具不同之地位：於唯識論藏層為《瑜伽師地論》卷二十二之六異門官方編定；於律宗註疏層為道宣《行事鈔》引《華嚴》三香之擴展；於天台止觀層則上昇為圓頓戒之般若深化，而其社會場域層之戒香表述讓予律宗。此一誠實揭露非批判智顗，乃天台與律宗教義分工之架構性確認：同一戒之教義於不同宗派體系中各承擔不同之功能位序，各宗於同一教義之不同層面深化，構成漢傳佛教教義分工之內生格局。本系列 P07 已記同一架構性缺位之模式——智顗《摩訶止觀》於八關齋戒一詞之命中數為零；P08 之戒香零命中為同一分工模式於本篇之延續確認：lay-realm 之週期出家機制（八關齋）與在家側日常持戒之社會場域效應（戒香）二者，於漢傳教義分工中皆讓予律宗註疏體系，而天台止觀層則上昇至圓頓戒之般若中觀層。

## 六、結論

本卷第二部自 P05 道之構成架構開卷，自 P06 五戒之經典範圍跨戒差別展開，自 P07 八關齋戒之週期出家機制具體展開，自本篇戒香之社會場域效應收束。第二部之四篇階梯結構由此完整——道之構成（P05)→ 戒禁內容（P06)→ 週期出家（P07)→ 社會場域效應（P08）。

第三部將以波羅提木叉（*pāṭimokkha*）為僧團內部之經典編定展開：P09 處理比丘戒結構之分疇式條目邏輯，P10 處理三系律藏比較性陳述。本篇戒香之社會場域效應於僧俗界面為外向表述，波羅提木叉為僧團內部之經典體制——二者為戒之社會場域效應之雙面框架，合構成戒於社會場域之內外兩面之完整展開。

戒香之社會場域可感知效應於戒至定之接續中具特殊地位。《增支部》第十一集第一經《何義經》之十一支鏈條前七支（戒 → 無悔 → 歡悅 → 喜 → 輕安 → 樂 → 定）為持戒至定之自然湧現之經典序列。本篇之核心主張為：**戒香之社會場域可感知效應為此十一支鏈條之在家側可感知入口**——在家眾於日常持五戒之社會場域顯現中，即令未進入止觀深修，亦已於發展性次第之第一支（戒 → 無悔）中行進。此一在家側可感知入口為 P24《楞嚴四清淨明誨》之戒至定接續之在家側預埋之另一直接經典機制，與 P07 八關齋之週期出家機制並列為在家側雙錨——前者（P07）為儀軌性週期高擡之入口，後者（P08）為日常持戒社會場域顯現之入口。二者於在家側相互補充，為戒至定之接續預備兩條經典路徑。

## 系列交叉引用

### 一、SĪLA 卷內前置篇章

- **P02《戒之預流支屬性》**：本篇引《增支部》第五集第三十二經 *Cundīsutta* 之預流支軸（*ariyakanta-sīla*，聖者所愛之戒）為旁證，P02 為主場。
- **P05《八聖道之三戒支》**：本卷第二部開卷篇，立正語 / 正業 / 正命為道之構成（path-elements），本篇承自其架構性語域。
- **P06《五戒之經典範圍》**：本卷第二部第二篇，立五戒之跨戒差別與酒戒之「護持道本身」性格，本篇 §三.4 酒戒社會場域可感知與十業道列位之分流為 P06 之直接 P08 進深。
- **P07《八關齋戒——一日一夜暫時出家之週期回返機制》**：本卷第二部第三篇，立八關齋戒為週期出家機制。本篇承自其架構性紀律——P07 智顗於八關齋零命中之架構性缺位於本篇戒香零命中為延續確認。

### 二、SĪLA 卷內後續篇章預埋

- **P09《波羅提木叉之架構》**：第三部開卷篇，處理比丘戒之分疇式條目邏輯。本篇戒香之社會場域效應於僧俗界面為外向表述，波羅提木叉為僧團內部之經典體制 embodiment——二者為戒之社會場域效應之雙面框架。
- **P10《三系律藏之對話》**：本篇 §五.5 之《根本說一切有部毘奈耶》卷五三蘊對應列舉為 P10 之主場保護，本篇僅作短引。
- **P12《大乘阿毘達磨集論》尸羅波羅蜜專章**：本篇 §五.3 之《大智度論》戒度章短引為 P12 之主場保護。
- **P16《戒體論》**：本篇 §五.3 之《優婆塞戒經》短引為 P16 之主場保護。
- **P17《道宣〈四分律行事鈔〉全展開》**：本篇 §三.2 + §五.5 之道宣《行事鈔》引《華嚴》三香短引為 P17 之主場保護。
- **P24《楞嚴四清淨明誨》**：本篇 §三.3 + §六之《增支部》11.1 十一支鏈條前七支引用為 P24 戒至定接續之在家側可感知入口之經典預埋。

### 三、跨論文重現錨點

- **AN 11.1 *Kimatthiyasutta***：本系列 P02（前 3 環預流支機制）+ P07（鏈式 ānisaṃsa 展開式）+ P08（戒蘊發展性次第）+ P24（戒至定承接聲明）之跨論文重現錨點，本篇為其發展性次第讀法之主要展開。

### 四、跨卷預埋（*nian*-volume + *dhyāna*-volume)

- **念卷 NIAN**：本系列 P02 預流支機制延承自念卷之「於戒不壞淨」之念之內生表述；本篇戒香之社會場域可感知效應為念之內生表述於日常社會場域之顯現面向。
- **定卷 SAMĀDHI**：本篇 §三.3 + §六之《增支部》11.1 十一支鏈條前七支引用為定卷之戒至定接續之 SĪLA 卷側預埋，於定卷將與《楞嚴四清淨明誨》之 monastic-grade *samādhi* 前行戒之承接形成雙錨。
