# 附錄 C　名相對照

正文裡，凡遇梵文、巴利的名相，多半只在首次出現時，用白話夾注一句，之後便直接用漢語詞，不再標注那些帶符號的羅馬字。完整的羅馬轉寫，連同巴利（巴）與梵文（梵）的對照，都收在這裡。

想知道某個詞的原文怎麼寫、它本來是什麼意思的讀者，可以查這張表。不想理會這些的讀者，跳過它，一點不影響讀這本書——這正是把它收在書後的緣故。

（標「巴」者為巴利，「梵」者為梵文；同一詞兩傳統寫法不同者並列。）

---

## 一、兩個骨幹詞，與它們的近鄰

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 願 | paṇidhāna / patthanā（巴）／ praṇidhāna（梵） | 把整個人的方向定下來、轉過來——本書的主角。許願添東西，發願轉方向。 |
| 決意 | adhiṭṭhāna（巴）／ adhiṣṭhāna（梵） | 立定不移的決斷——南傳十波羅蜜之一；願之「不退」一面的巴利根。 |
| 誓 | （願之加固式） | 把願說死、押上代價的那一種發法——「設我得佛……不取正覺」的句式。 |
| 迴向 | pariṇāmanā（梵） | 把功德的朝向轉出去——回謂回轉，向謂趣向；轉的是朝向，不是物。 |
| 發（菩提）心 | bodhicittotpāda（梵） | 發起「上求佛道、下化眾生」的根本之心——本書所講「發願」的正式名。 |

## 二、想要這股力

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 欲（中性的機能） | chanda（巴／梵） | 心對著所喜之境伸出去要的那股力——本身無善惡，看朝向。 |
| 善法欲 | kusala-dhamma-chanda（巴） | 朝向善法、朝向解脫的想要——道的引擎，要養；到了地方自己會熄。 |
| 渴愛 | taṇhā（巴）／ tṛṣṇā（梵） | 得了還要、永不竭盡的攀取——生死的根，要斷。 |
| 無願（三昧） | appaṇihita（巴）／ apraṇihita（梵） | 三解脫門之一：於一切法不再以佔有的方式願求——願的階梯最高一格，而本願不斷。 |

## 三、願在菩薩道上的名位

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 波羅蜜 | pāramī（巴）／ pāramitā（梵） | 到彼岸。願是十度第八席——「尚未到彼岸」的那一種波羅蜜。 |
| 願波羅蜜（多） | praṇidhāna-pāramitā（梵） | 悕求後後智——永遠探身向前、替當來牽引勝緣的那一度。 |
| 十大願（初地） | （歡喜地十願） | 初地菩薩登地第一件事所發的十願——十地長路的骨架，以十盡句封頂。 |
| 十盡句 | （若眾生界盡，我願乃盡……） | 把願的終點，刻意設在永不成立之處——願以永不關門為本分。 |
| 四弘誓願 | （漢傳通途菩薩願） | 眾生無邊誓願度，煩惱無盡誓願斷，法門無量誓願學，佛道無上誓願成——四聖諦的主動式。 |
| 無作四弘 | （天台判） | 明知眾生如空、誓度如空之眾生的那種發法——無作不是不作，是做而不執完成。 |

## 四、本願與他力

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 本願 | pūrva-praṇidhāna（梵） | 宿世根本、已經成就、現在就具足效力的願——「本」不是「他發的」，是「已成的」。 |
| 四十八願 | （法藏比丘所發） | 每一條以「不取正覺」收尾的四十八條——一份生死攸關的擔保書，不是設施清單。 |
| 他力 | （曇鸞所判） | 乘一個已成就之願的力而行——交付的是方向的承載，不是方向本身；不是偷懶，不是買賣。 |
| 作願門 | （五念門之三） | 行者這一端日日發起的願——「心常作願，一心專念，畢竟往生安樂國土」。 |
| 往相・還相 | （二種迴向） | 往：迴向己德願生彼土；還：生彼土已，迴入生死稠林度眾生——上了岸的人是要再下水的人。 |

## 五、行願與相續

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 行願 | bhadracarī-praṇidhāna（梵，普賢行願） | 行與願連成的一個詞——願是行之所朝向，行是願之所填實；如鳥二翼，如車二輪。 |
| 十大願王 | （普賢十願） | 禮敬諸佛乃至普皆迴向——與初地十願是兩套；晚課所誦者即此。 |
| 念念相續・無有疲厭 | （十願套句） | 願活在時間裡的方式——持線的是觀行之力，不是那個會疲的我。 |
| 觀行力 | （宗密語） | 願自己在走的那股力——「若不以普賢觀行之力，如何得念念不斷？」 |
| 普皆迴向 | （第十願） | 前九是積，第十是轉——放下的是會計師，不是功德。 |

## 六、願與空

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 空 | suññatā（巴）／ śūnyatā（梵） | 不是「沒有」，是「無一個攥得住、鐵打的自性」——願最深的盟友，不是對手。 |
| 無所得 | anupalambha（梵） | 不可得——不是減法；無所得而發的願，發得更鬆，不是發得更少。 |
| 三輪體空（於願） | （比照 trimaṇḍala-pariśuddhi） | 願者、所願、願三者皆不可得——此係比照布施三輪之結構投射，經中無逐字成句，正文已明示。 |
| 無生法忍 | anutpattika-dharma-kṣānti（梵） | 於「諸法本不曾生」忍可安住——不退的真正保證：無一法生起以供失去。 |
| 不退轉／阿鞞跋致 | avaivartika（梵） | 願不可奪、不可竭之位——以無地基為不動，非以抓得最緊為不動。 |
| 不動地 | acalā bhūmi（梵） | 第八地——成就無生法忍即時得入；捨功用行，得無功用法。 |
| 無功用 | anābhoga（梵） | 不假勉強、任運自然——不動不是用功的頂點，是用功的脫落。 |
| 無住處涅槃 | apratiṣṭhita-nirvāṇa（梵） | 般若故不住生死，大悲故不住涅槃——能走而不走，才叫留。 |

## 七、終點

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 阿耨多羅三藐三菩提 | anuttarā samyak-saṃbodhiḥ（梵） | 無上正等正覺——《心經》末後那個「得」字所得者；圓成而無人圓成。 |
| 究竟涅槃（斷果） | （法藏判） | 障盡之寂滅——說「什麼已經沒有了」的成就；菩薩一句的落點。 |
| 得菩提（智果） | （法藏判) | 智圓之具足——說「什麼已經圓滿現前了」的成就；諸佛一句的落點。後者圓成前者，不可混同。 |
| 一切智海 | sarvajñatā（梵，一切智） | 諸佛無邊的智慧之海——「從如來往昔大願之所生起」：佛果由願生、由願現。 |
| 五分法身 | （戒・定・慧・解脫・解脫知見） | 懺淨之後「生」出的新所依——無我地基上的清淨相續，不是換一個新的我。 |
