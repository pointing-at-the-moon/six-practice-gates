# 第九部 · 現代回歸 · 今天的人，出場了

八部走完，定的兩極立穩了：內收到一念離相，外延到法界齊現。可是這一路上，有一個身影始終沒有出場——今天的人。

定並沒有死在經卷裡。它換了一身衣裳，回到了千千萬萬人的生活中：手機裡的計時器，公司裡的減壓課，房間裡那張安靜的蒲團。定回來了——而它一回來，就帶回了兩個古老的危險，只是換了現代的面孔。

第九部三章。

第一章，兩種失格。把定當成隨手取用的紓壓工具，是一種失格；用定躲開該面對的事、把定中之境誤認作證量，是另一種失格。看似相反，病根同一——定離了慧。這兩種病，傳統早有名字，也早開好了藥。

第二章，AI 時代之定。當身邊的機器以爭奪注意為業，「把心收成一點」這件事，遇上了一個古人沒有的新外緣。它是定的敵人嗎？是定的幫手嗎？兩個問法都問錯了——錯在哪裡，經論裡有一把現成的尺。

第三章，全卷的末章——內向之門回開。本卷從佛陀捨無色定的那一幕開篇；走過三十章，要回到那一幕，把種在卷首的話結成果：定不是終點，定終須回歸般若。

先立一條本部之約：這三章只診病，不開方。它是一面鏡子，不是一份功課。

照鏡子吧。
