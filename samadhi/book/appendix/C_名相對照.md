# 附錄 C　名相對照

正文裡，凡遇梵文、巴利的名相，多半只在首次出現時用白話夾注一句，之後便直接用漢語詞，不再標注那些帶符號的羅馬字。完整的羅馬轉寫，連同巴利（巴）與梵文（梵）的對照，都收在這裡。

想知道某個詞的原文怎麼寫、它本來是什麼意思的讀者，可以查這張表。不想理會這些的讀者，跳過它，一點不影響讀這本書——這正是把它收在書後的緣故。

（標「巴」者為巴利，「梵」者為梵文；同一詞兩傳統寫法不同者並列。）

---

## 一、三個骨幹詞

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 定／三昧／三摩地 | samādhi（巴／梵） | 心安住於一而不散——本書真正的主角；音譯作三昧、三摩地。 |
| 止／奢摩他 | samatha（巴）／śamatha（梵） | 令心寂止的那一面工夫——與觀同負一軛的一條軌道。 |
| 禪那／靜慮 | jhāna（巴）／dhyāna（梵） | 四禪的「禪」——一級一級可以指認的定境；漢地「禪宗」之禪，另是第三義。 |

## 二、巴利根基與論書的詞

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 觀／毘鉢舍那 | vipassanā（巴）／vipaśyanā（梵） | 如實照見的那一面工夫——止觀雙軌的另一條。 |
| 心一境性 | cittass'ekaggatā（巴） | 心專注於一境——定的體；全卷那個「一」字。 |
| 尋／伺 | vitakka／vicāra（巴） | 心投向所緣的粗動與細轉——初禪有、二禪捨。 |
| 喜／樂 | pīti／sukha（巴） | 禪支裡的歡騰與安樂——三禪捨喜，四禪連樂也放下。 |
| 捨 | upekkhā（巴）／upekṣā（梵） | 不偏不倚的平等持心——四禪所餘的純淨一念；永嘉「優畢叉」即此。 |
| 業處 | kammaṭṭhāna（巴） | 修定的下手對境——四十扇門的「門」。 |
| 近行定／安止定 | upacāra／appanā（巴） | 將入未入的門口之定，與全然安住的根本之定。 |
| 取相／似相 | uggaha-nimitta／paṭibhāga-nimitta（巴） | 所緣在心中內化的兩個階段。 |
| 五蓋 | pañca nīvaraṇāni（巴） | 蓋住定的五位常客——貪欲、瞋恚、惛沉、掉悔、疑。 |
| 滅盡定 | nirodha-samāpatti（巴） | 聲聞所證最深的定——維摩丈室裡作對照的那一座。 |
| 正定 | sammā-samādhi（巴） | 八正道第八支——編在道上、朝著慧的四禪。 |

## 三、大乘與般若的詞

| 書中用語 | 羅馬轉寫 | 一句白話 |
|---------|---------|---------|
| 禪波羅蜜 | dhyāna-pāramitā（梵） | 六度之一——為利益眾生而修的定。 |

| 法愛 | dharma-rāga（梵） | 對善法、對證得的微細愛著——於人天為妙，於無生法忍為累。 |

| 一行三昧 | ekavyūha-samādhi（梵） | 繫緣法界一相之定——慧為定門。 |
| 首楞嚴 | śūraṅgama（梵） | 健行、健相之義——含攝諸三昧的定中之王。 |

| 因緣 | hetu-pratyaya（梵） | 親辦自果的直接生因——定的因緣，唯行者自心。 |

## 四、幾個漢地自家的詞

這幾個詞是漢語佛教自己長出來的，沒有梵巴原文可對；一併收在這裡，給一句白話。

| 書中用語 | 一句白話 |
|---------|---------|
| 壁觀 | 達摩法門的招牌——承重不在牆，在「心如牆壁」的安心。 |
| 安心 | 達摩給「定」的新名字——覓心了不可得處，安心已竟。 |
| 默照 | 宏智的家風——默（止）與照（觀）同時現成，缺一即非。 |
| 話頭 | 話之「頭」——一念未生之際；看話頭即迴光返照能起念者。 |
| 疑情 | 「能起念的畢竟是誰」之整體迫切——吞不下吐不出的熱鐵丸。 |
| 味禪／闇證 | 天台對定脫離慧的雙診——耽溺定樂，與昏闇中誤認證量。 |
| 作聖解 | 把定中過渡之境執定為聖證——五十陰魔全圖的那個開關。 |
